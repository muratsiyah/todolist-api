# Todolist-API
This project is a spring boot api like Wunderlist and Todoist.

## Technology Stack
- Java 8
- Spring Boot
- Spring Data Couchbase
- Couchbase
- Swagger
- Maven
- Docker
- JUnit and Mockito

## Run Tests
- mvn clean test

## Run Project
- docker-compose up (Docker compose must be installed on the computer.)
